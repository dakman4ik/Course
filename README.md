# Course 

Реализация собственного динамического контейнера (массива) на С++ с ручным управлением памятью и поддержкой математических операций.



# Описание
Представлен лишь класс контейнера с тестами.
* Правило трёх: конструктор копирования, оператор присваивания, деструктор.
* Автоматическое увеличение ёмкости (`capacity`) в 2 раза, при нехватке места (`rellocate`). 


# Базовые операции с контейнером

`push_back(int)` -  Добавление элемента в конец.

`pop_back()` - Удаление последнего элемента.

`insert(int a, size_t index)` - Вставка по индексу (С проверкой границ, кидает `std::out_of_range`).

`erase(size_t index)` - Удаление по индексу (С проверкой границ, кидает `std::out_of_range`).

`count(int)` - Подсчёт кол-ва определённого значения в массиве (-1, если не найдено).

`get_size()` / `get_capacity()` - Текущий размер / ёмкость контейнера.

`operator[]` - Доступ к элементу (С проверкой границ, кидает `std::out_of_range`).





# Ввод/Вывод

* `operator>>` — чтение количества элементов и самих элементов из потока.
* `operator<<` — вывод элементов через пробел.



# Арифметические операции

Поддерживаются операции сложения, вычитания, умножения и деления:

* Поэлементно между двумя объектами `Course`.
* Между объектом `Course` и `int` (в обе стороны).
* Как в форме составного присваивания (`+=`, `-=`, `*=`, `/=`), так и в форме бинарного оператора, возвращающего новый объект (`+`, `-`, `*`, `/`).
